# 💳 Credit Scoring Model using Random Forest

## 📌 Project Description

This project uses the **Random Forest** algorithm to predict whether a loan applicant is a **good or bad credit risk** based on financial and personal information. The model helps support credit risk assessment and loan approval decisions.

## 📊 Dataset Description

- **Dataset Name:** German Credit Scoring Dataset
- **Source:** Kaggle / UCI Machine Learning Repository
- **Records:** 1,000 loan applicants
- **Features:** 20 input features and 1 target variable
- **Problem Type:** Binary Classification
- **Target Variable:** Credit Risk (Good / Bad)
- **Missing Values:** No missing values

## 🎯 Objective

To build a machine learning model that accurately classifies applicants as **Good** or **Bad** credit risks, assisting financial institutions in making informed lending decisions.
